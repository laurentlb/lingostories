Syntax Cheat Sheet

Need help?

  • Read LingoStories writing guide.
  • Read full Ink documentation for comprehensive guides and examples. Remember that the text needs to be translated, so you cannot include variables as part of the text.

Below is a quick cheat sheet.

Choices

  • * choice that can be chosen only once
  • + sticky (can be chosen again)

  • Merge flow after a set of choices with a gather line -.

* Go north
   You see mountains.
* Go south
   You see the sea.
-
You keep walking.

Knots and Stitches

  • Knots (top-level sections)
=== forest ===
  • Stitches (subsections inside knots)
= clearing =

Diverts

  • Jump to another knot or stitch
-> forest
  • End story
-> END

Brackets

Use square brackets after a choice, if the choice is not part of the story.

+ [Behind a big tree.]
   You decide to hide behind a big tree.
   -> hidden

Comments

// This is a single line comment

/* This is a multiline
line comment */

Dialogues

Prefix a sentence with @name if a character says something.

@tom Hi Anna, it's great to see you here!

Text variations

When the story has loops, it can feel repetitive to always see the same text. It's possible to have some variations (see documentation): 

// Cycle through options.
{&It was the morning.|It was the afternoon.|It was the evening.}
// Go through the options only once.
{!I laughed.|I smiled.|I grimaced.|I ignored him.}
// Randomly pick one option.
I tossed the coin. {~Heads.|Tails.}

Variables

  • Declare and assign
VAR health = 10
~ health = health - 1

Conditionals

  • Conditional choices
* {health > 0} Keep fighting
* Resign
